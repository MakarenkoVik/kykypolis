    1. Анализ бизнеса.
Для лучшего понимания структуры, политики и операций бизнеса был проведен бизнес-анализ. Предварительно собраны данные по возможностям и желаниям заказчика.
Была выявлена необходимость увеличения уровня ликвидности оборотных средств, а именно: уменьшение отсрочек платежа у уже существующих клиентов и привлечение 
новых клиентов (физических лиц), которые вносят предоплату за оказанные услуги. Упор был сделан на часть бизнеса по предоставлению услуг, более детально - 
на семьи с детьми и на компании, которые будут пользоваться услугами, предварительно оплатив за них. Уровень проникновения интернета сейчас составляет около 
60% мирового населения и около 67% населения пользуется мобильными устройствами. Соотвественно, один из лучших вариантов, для привлечения новых клиентов и 
удержания старых - это запуск одностраничного сайта услуг для организации с поддержкой отображения на мобильных устройствах.

    2. Переговоры с заказчиком.
С помощью опросного листа была подтверждена гипотеза о необходимости привлечение клиентов через обособленную чать бизнеса по услугам агропарка, что позволит
саккумулировать больше наличных денежных средств в кассе предприятия, повысить эффективность и качество оказываемых услуг с помощью обратной связи и привлечет
новых партнеров и средства массовой информации для дополнительной рекламы.

    3. Составление техзадания.
1. Общая информация о проекте
Проект - kykypolis.
Сайта прежде не было.
Планируемый адрес сайта - https://kykypolis.by.  
Тип сайта - одностраничный сайт.
Необходима ли версия сайта на другом языке - да(English).

2. Структура сайта
1. Первая часть - то, что посетитель видит при открытии нашей страницы, не используя скроллинг (требуется заинтересовать клиента, заголовок, картинка, эмоция);
1.1. Header (логотип, меню, контакты, переключатель языков);
1.2. Основная картинка с призывом;
2. Вторая часть - то, что идет ниже первого экрана, прокрутив колесико мыши (выгоды, возражения, факты);
2.1. Описание парка;
2.2. Перечень услуг с ценами;
2.3. Факты;
2.4. Галерея;
2.5. Подписаться на новости;
2.6. Новости;
2.7. Отзывы;
2.8. Контакты с картой;
2.9. Партнеры;
2.10. Footer(копирайт, меню сайта, контакты)
3. Третья часть - регистрация клиента (обратная связь).

3. Административная панель.
У менеджера есть возможность:
- добавлять, удалять и корректировать существующие услуги;
- добавлять, удалять и корректировать существующие цены;
- добавлять, удалять и корректировать существующие фотографии;
- добавлять, удалять и корректировать существующие отзывы;
- добавлять, удалять и корректировать существующие события;
- добавлять, удалять и корректировать существующие пользователей.
